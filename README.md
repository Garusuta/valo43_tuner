# VALORANT 4:3 分辨率自动切换器

> 无畏契约游戏启动时自动切换到4:3分辨率，关闭时自动恢复原始分辨率

## 🎮 功能特点
- 自动检测无畏契约启动/关闭
- 游戏启动时自动切换到4:3分辨率
- 游戏关闭时自动恢复原始分辨率
- 支持自定义分辨率和刷新率

## ⚙️ 配置文件说明

首次运行会在程序目录下生成 `config.toml` 配置文件：

### 配置文件格式
```toml
[Desktop]
ResolutionSizeX = 1920    ; 桌面原始分辨率 - 宽度
ResolutionSizeY = 1080    ; 桌面原始分辨率 - 高度
RefreshRate = 144         ; 桌面刷新率

[Game]
ResolutionSizeX = 1568    ; 游戏内分辨率 - 宽度（4:3拉伸）
ResolutionSizeY = 1080    ; 游戏内分辨率 - 高度
RefreshRate = 144         ; 游戏内刷新率
```

### 如何修改配置
1. 关闭程序
2. 用记事本打开 `config.toml`
3. 修改对应的数值：
   - **Desktop** 部分：设置你的桌面原始分辨率
   - **Game** 部分：设置你想要的无畏契约4:3分辨率
4. 保存文件并重新启动程序

### 刷新率说明
- 确保 `RefreshRate` 与你的显示器支持的刷新率一致

## 🚀 使用方法
1. 右键程序 → "以管理员身份运行"
2. 程序会自动在后台运行
3. 启动无畏契约 → 自动切换到4:3分辨率
4. 关闭游戏 → 自动恢复桌面分辨率

## ⚠️ 注意事项
- 首次运行**必须**使用管理员权限
- 修改配置后需要重启程序生效
- 确保显示器支持设置的分辨率和刷新率
- 如果切换失败，请检查是否添加了自定义分辨率
- 目前仅支持国服
